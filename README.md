# VidyaHub 🎓

VidyaHub is a comprehensive educational platform designed to provide students with high-quality, NCERT-aligned learning resources. It integrates interactive games, video lessons, and AI-powered assessments to create an engaging learning environment.

## 🚀 Features

- **NCERT Aligned Content**: Video lessons and quizzes tailored for classes 1-12.
- **Gamified Learning**: Earn XP, level up, and complete achievements through games like:
  - Memory Match
  - Word Scramble
  - Space Defender
- **Boss Battles**: Challenging quizzes to test mastery of subjects.
- **AI-Powered Assessments**: Intelligent MCQ generation and progress tracking.
- **Student & Teacher Dashboards**: Dedicated interfaces for tracking progress and managing content.

## 🛠️ Technology Stack

- **Backend**: Python (Django)
- **Frontend**: HTML5, Vanilla CSS (Glassmorphic Design), JavaScript
- **Database**: SQLite (Development)
- **Real-time**: WebSockets (Django Channels)

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Satyam3se/VidyaHub.git
   ```
2. Navigate to the project directory:
   ```bash
   cd VidyaHub
   ```
3. Install dependencies (make sure you have Python installed):
   ```bash
   pip install -r requirements.txt
   ```
4. Run migrations:
   ```bash
   python manage.py migrate
   ```
5. Start the development server:
   ```bash
   python manage.py runserver
   ```
